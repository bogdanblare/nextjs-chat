### Nextjs-chat
# Run Locally

Clone the project

```bash
  git clone https://github.com/bogdanblare/nextjs-chat.git
```

Go to the project directory

```bash
  cd nextjs-chat
```

Install dependencies

```bash
  pnpm install
```

Start the server

```bash
  pnpm run dev
```
